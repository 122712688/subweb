使用指南：
安装python3 依赖
apt-get update -y
apt install -y python3-pip  git python3

安装库：
pip3 install requests
pip3 install flask
pip3 install flask_restful

修改文件：
修改 config/subconverter   的文件权限，全部勾选
修改 api/aff/py            为自己的ip

开始运行（已screen为例）：
第一个screen
screen -R web
cd /root/subweb
python3 api.py

再建一个screen
screen -R sub
cd /root/subweb/config
./subconverter
